# 📊 Power BI Sales Dashboard

An interactive **Power BI Dashboard** created to analyze and visualize sales performance using multiple datasets.  
This dashboard provides insights into sales trends, supervisor performance, and product-wise revenue.

---

## 🚀 Project Overview

This project focuses on transforming raw sales data into meaningful visual insights using **Power BI**.  
It helps in:

✅ Monitoring overall sales performance  
✅ Identifying top-performing supervisors  
✅ Understanding product-wise and region-wise sales trends  
✅ Supporting data-driven business decisions  

---

## 🗂️ Dataset Used

The dashboard was built using the following datasets:

📁 `Sales_data.csv` → Contains transactional sales information  
📁 `Sales_list.csv` → Contains product and sales details  
📁 `Supervisor.csv` → Contains supervisor related information  

---

## 📈 Dashboard Features

✨ Interactive filters and slicers  
✨ KPI cards for quick performance overview  
✨ Sales trend analysis  
✨ Supervisor performance comparison  
✨ Product/category level insights  
✨ Clean and user-friendly visualization
